# Security Policy

## Supported Versions

We actively monitor and patch security vulnerabilities in the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0.0 | No        |

---

## Reporting a Vulnerability

We take the security of **AI Gateway Pro** seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, report it directly to our security team.

### How to Report
1. Email your report to [email protected].
2. Include a detailed description of the vulnerability, steps to reproduce, and any proof of concept (PoC) code.
3. We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue under coordinated disclosure guidelines.

Thank you for helping keep AI Gateway Pro secure!
